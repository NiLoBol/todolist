# โปรเจคToDoList

โปรเจคTodolist ที่สามารถ เพิ่มข้อมูล ลบข้อมูล และแก้ไขข้อมูล โดยใช้ localStorage ในการเก็บข้อมูลภายใน

## เริ่มต้น

ดาวน์โหลดไฟล์ todolist และติดตั้ง

### การติดตั้ง
ติดตัง modules
```bash
npm i
```
ใช้งานด้วย
```bash
npm start

```
